## 개요

솔라가드 건축용 필름 브랜드의 보증서 관리 어드민과 조회 서비스입니다. 사전 개발해둔 Next.js 어드민 프레임워크를 기반으로 제작하였고, 조회 페이지와 어드민, API, 데이터 구조, 배포 환경까지 포함한 전체 제품 범위를 단독으로 설계·구현했습니다.

기존 파노라마 필름 등의 프로젝트에서의 PDF 생성 기능 개발 경험을 바탕으로, 2주 안에 운영 가능한 보증서 발급 시스템으로 런칭했습니다.

## 작업범위

- 보증서 조회 사이트 UI 디자인 및 개발
- 보증서 발급 관리 어드민 개발
- react-pdf 기반 보증서 렌더링 컴포넌트 구현
- 보증서 조회·PDF 발급 API와 조회 플로우 구현
- 엑셀 대량 발급 플로우 및 운영자 교정 UI 구현
- Supabase 기반 데이터 구조, 서버 액션, 배포 환경 구성

## 주요 기능

- 보증서 정보 기반 조회
- 보증서 PDF 미리보기·인쇄·다운로드
- 보증서 발급 및 발급 내역 관리
- 엑셀 기반 보증서 대량 발급
- 보증 제품 옵션·시공점·보증기간 관리

## 핵심 설계 결정

### 보증서 렌더링 스택 선정

기존 프로젝트에서 경험한 html2pdf.js, Puppeteer 기반 PDF 생성 방식의 문제점을 react-pdf 라이브러리가 해결할 수 있다는 점을 파악하였습니다. 가볍고 디자인 구현도 충분하며, 문서 규격을 맞추기도 쉽고 고해상도 PDF 생성도 가능했습니다.

### 보증서 렌더링 경로 단일화

보증서 렌더링 컴포넌트는 어드민 프로젝트에 하나만 두고, 조회 페이지는 어드민의 API 라우트를 통해 렌더링 결과를 받아 쓰도록 구성했습니다.

### 보증서 조회 사이트 경량화

보증서 조회 페이지는 보증서 조회와 PDF 열람에 필요한 프론트엔드만 담당하도록 구성했습니다. 보증서 원본 데이터와 PDF 생성 로직은 어드민 앱에 두고, API 호출을 통해 결과를 표시하는 구조로 분리했습니다.

### 엑셀 대량 발급 워크플로우 설계 및 구현

관리자가 엑셀을 통해 보증서를 대량으로 발급할 수 있는 기능입니다. 데이터를 서버에 요청하기 전, 화면에서 데이터 검증을 할 수 있도록 플로우를 구현했습니다.
