## Overview

기존 PHP 기반 프로젝트 관리툴을 Next.js·MongoDB 기반 사내 운영 시스템으로 개편한 프로젝트입니다. 사내 20명 이내가 실제 업무에 사용했습니다.

## 작업범위

- 별도 디자인 없이 업무 흐름을 직접 파악해 UI·기능 설계
- 프론트엔드·관리 기능 개발
- 기존 MySQL 데이터의 MongoDB 마이그레이션

## Key Features

- 프로젝트·구성원·포트폴리오 관리
- 고객·문의 관리
- 견적서 관리와 견적서 PDF 출력

## Challenges

### 게시판형 기록 도구를 운영 시스템으로

기존 도구는 게시판 형태의 프로젝트 기록에 가까워, 프로젝트 현황과 구성원·포트폴리오·고객·문의·견적서를 함께 관리하기 어려웠습니다.

프로젝트·구성원·포트폴리오·문의·고객·견적서 관리와 견적서 PDF 출력을 하나의 흐름으로 묶었습니다.
