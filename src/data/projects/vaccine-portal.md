## Overview

백신 연구개발·상용화에 필요한 지원시설, 행사, 교육 정보를 제공하는 데이터 포털과 관리자 CMS입니다.

## 작업범위

- PC 디자인을 제외한 반응형 UI 구현
- 프론트엔드 개발
- 관리자 CMS
- DB 구조 설계와 데이터 정규화
- 프로젝트 기획·관리

## Challenges

### 여러 카테고리에 걸친 시설 데이터, 복합 탐색 구조로

한 시설이 비임상·임상·위탁생산 등 여러 카테고리에 동시에 속하고, 카테고리마다 상세 속성과 필터 조건이 달라 단순 게시판이 아닌 **복합 데이터 탐색 구조**가 필요했습니다.

지원시설 데이터를 정규화하고, 다음과 같은 조건으로 탐색 가능한 정보 구조와 UI를 설계했습니다.

1. 국내/해외, 지역
2. 시설 구분(비임상·임상·위탁생산)
3. 생물안전등급, 인증 현황
