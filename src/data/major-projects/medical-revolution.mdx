## 프로젝트 개요

(주)엠알의 홈페이지 개편 / 홈페이지 기획, 설계 및 개발 진행.

## 기술 선정 사유

- <code>**Next.js**</code> : SEO 최적화 / 인터렉션 구현 용이성 / 1인 개발의 용이성
- <code>**Tailwind CSS**</code> : 빠른 디자인 구현 및 일관성 유지
- <code>**framer-motion**</code> : 스크롤 인터렉션 구현
- <code>**shadcn/ui**</code> : form 컴포넌트 및 어드민 UI 구현
- <code>**next-auth**</code> : 회원 인증 기능 구현
- <code>**nodemailer**</code> : SMTP를 통한 이메일 발송 기능 구현
- <code>**Supabase**</code> : 빠른 백엔드 환경 구축 및 데이터베이스 개발, 고객사의 운영 관리 부담 절감

## 주요 작업 내역

- 홈페이지 기획, 설계 및 프로젝트 관리
- 메인페이지 스크롤 인터렉션의 기획
- PC 화면 디자인을 기반으로 한 반응형 디자인 구현
- 홈페이지 기획 및 디자인 시안 분석을 통한 재사용 UI 컴포넌트 설계
- 회원 전용 교육영상 인증 레이어 구현
- 게시판 및 문의 (인바운드) 기능 구현
- 어드민 페이지 구축 / 데이터베이스 구축
