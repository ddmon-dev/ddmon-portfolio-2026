# 프로젝트 개요

백신실용화기술개발사업단의 백신 연구개발 정보 포털 사이트의 기획, 설계 및 개발 진행

## 기술 선정 사유

- <code>**Next.js**</code> : 포털 사이트의 SEO 최적화 및 성능 최적화, 1인 개발 효율성
- <code>**CSS Modules**</code> : 포털 웹사이트의 디자인 구현 및 유지보수의 용이성
- <code>**Tailwind CSS**</code> : 어드민 페이지의 일관되며 빠른 디자인 구현 효율성
- <code>**FullCalendar**</code> : 행사 일정 관리 기능 구현
- <code>**Supabase (Self Hosted)**</code> : 빠른 백엔드 환경 구축 및 데이터베이스 개발
- <code>**Nginx**</code> / <code>**Nginx-Proxy-Manager**</code> / <code>**Ubuntu**</code> : 고객사 사정으로
  인한 해외 서비스 사용 불가로 자체 운영 서버 구축

## 주요 작업 내역

- 포털 사이트의 기획 및 설계
- 약 300개 가량의 지원시설데이터의 분석, 분류 및 데이터베이스 구축
- 포털 사이트의 반응형 최적화 디자인 구현
- 지원시설데이터 상세 검색기능 구현
- 행사 일정 캘린더 구현
- 교육 일정 게시판 구현
