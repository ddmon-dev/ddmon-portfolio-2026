## 프로젝트 개요

그누보드5 기반 자사 수주 홈페이지 개발 프레임워크 구축

## 기술 선정 사유

- <code>**GNUBOARD 5**</code> : 기존 홈페이지 제작 프로세스와의 호환성을 유지, 개발 효율성 증대
- <code>**PHP**</code> : 고객사 시장 여건 고려하여 호스팅 용이성 고려 (Cafe24, 가비아 호스팅 등)

## 주요 작업 내역

- 홈페이지 제작시의 요구사항 및 기능 패턴 분석
- 홈페이지 주요 레이아웃 및 구조 패턴 분석
- 내부 개발 인력 풀 고려한 프로젝트 템플릿 구조 설계
