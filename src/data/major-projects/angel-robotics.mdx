## 프로젝트 개요

엔젤로보틱스 홈페이지의 3번째 개편 작업 진행. (1차 2022년 개편, 2차 2023년 개편)

## 기술 선정 사유

- <code>**Next.js**</code> : SEO 최적화 / 인터렉션 구현 용이성 / 1인 개발의 용이성
- <code>**Tailwind CSS**</code> : 빠른 디자인 구현 및 일관성 유지
- <code>**framer-motion**</code> : 스크롤 인터렉션 구현
- <code>**shadcn/ui**</code> : form 컴포넌트 및 어드민 UI 구현
- <code>**Kakao Map API**</code> : 도입 병원 찾기 서비스 구현
- <code>**Supabase**</code> : 빠른 백엔드 환경 구축 및 데이터베이스 개발, 고객사의 운영 관리 부담 절감

## 주요 작업 내역

- framer-motion을 활용한 스크롤 인터렉션 구현.
- PC 화면 디자인을 기반으로한 반응형 디자인 구현.
- 어드민 페이지의 개편 작업 진행 (PHP -> Next.js)
- 기존 홈페이지 데이터의 개편 및 이전 (MySQL -> Supabase)
